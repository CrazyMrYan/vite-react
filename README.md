# vite-react

使用 [create-app-template-react](https://github.com/vitejs/vite/tree/main/packages/create-app/template-react) 模版

## 克隆
```shell
git clone git@github.com:CrazyMrYan/vite-react.git
```

## 下载依赖
```shell
yarn install
```

## 启动
```shell
yarn dev
```

## 使用技术
- react 
- vite
- antd
- react-router-dom
- @ant-design/charts
- @ant-design/icons
- pubsub-js
